### Environment setup

- [Anaconda](https://www.anaconda.com/), for environment management.

```
    conda env create -f environment.yaml
    conda activate sphere-mapping
```

### Usage

- CLI

```
    python main.py -h
    python main.py -m uniform_area
```

- Or [Notebook](graph.ipynb).
